Proxy exclusion setting

Procedure

This document explains how to set up proxy for use with Google Chrome.

  1. Click "Settings" - "Advanced" - "System" - "Open proxy settings".

  2. Click "Internet Properties" - "Connection" - "LAN settings".

  3. Uncheck "Use automatic configuration script".

  4. Check "Use proxy server for LAN" and enter the following.

    Address：{ProxyName}  
Port   ：8080

  5. Check "Not use proxy server for local address".

  6. Click the "Advanced" button.

  7. Check "Use the same proxy server for all protocols".

  8. Enter "personium.example.com" in the text box "Not use proxy for addresses beginning with next," and Click "OK" button.

  9. Click "OK" in all windows to close the setting screen.
