Overview

write

    • Unpublished

/{CellName}/{BoxName}/{CollectionName}/__src/{ResourceName}

{CellName} Cell Name
{BoxName} Box Name
{CollectionName} Service Collection Name Valid values
Number of digits:1-128
Usable character types
alphanumeric character, period(.), under score(_), hyphen(-)
{ResourceName} Resource name Valid values(limit)
Number of digits:1-128

alphanumeric character, period(.), under score(_), hyphen(-)
